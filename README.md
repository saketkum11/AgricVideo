# AgricVideo

#### AgricVideo is a video list that has a variety of videos with new technology related to agriculture.

# Tech Stack
  - React JS
  - Html
  - Webkit-ui
  - Mockbee Backend
  
  
 

# Features 
  - Login 
  - Signup
  - History Video
  - Watchlater Video
  - Liked Video
  - Create Playlist 
  - Single Video
